# Rooftop Garden

A cozy farming game built with Godot 4.5, inspired by Stardew Valley.

![Screenshot](misc/screen.png)

## Requirements

- [Godot 4.5](https://godotengine.org/download) or later

## Getting Started

1. Clone the repository
2. Open `project.godot` in Godot
3. Press F5 to run the game

## Project Setup

- **Resolution:** 720x1280 (portrait mobile)
- **Rendering:** Mobile renderer with pixel art filtering
- **Stretch Mode:** Canvas items (responsive scaling)
